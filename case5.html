400여 개 제휴 골프장 정책 표준화 → 상담 기준 통일

카카오VX에서 취소 정책·위약금 기준이 제각각인 골프장 파트너 400여 곳의 정책을 직접 수집·기준화해, 상담사 전원이 동일한 판단 기준으로 응대할 수 있는 운영 체계를 처음부터 설계한 사례입니다.
카카오VX (파견) B2B 파트너 운영 정책 표준화 운영 가이드 재정비 상담 기준 통일

Problem

파트너별 기준이 달라 오안내와 반복 문의가 지속됐습니다

400여 개 골프장, 정책이 제각각
제휴 골프장마다 취소 정책, 위약금 기준, 동반자 제한 규정이 달랐습니다. 파트너사 어드민(카카오파트너스)에 데이터가 존재했지만, 기준화되지 않은 상태여서 상담사마다 다르게 안내하는 상황이 반복됐습니다.
동일 유형 문의 반복 + 오안내 컴플레인
파트너로부터 동일한 유형의 문의가 반복 유입됐고, 기준 혼선으로 인한 오안내·파트너 컴플레인이 지속됐습니다. 어드민 데이터의 오기재·미입력 항목도 상담 혼선의 원인 중 하나였습니다.

Action

정책 기준화부터 운영 가이드 배포까지 직접 설계했습니다

Action 01
400여 개 파트너 정책 직접 수집 및 기준화
취소 정책, 위약금 기준, 동반자 제한 등 파트너별 핵심 정책 항목을 중요도 순으로 직접 재구조화했습니다. 어드민 데이터를 건별로 검토해 오기재·미입력 항목을 직접 수정 처리해 데이터 정합성을 확보했습니다. 파트너 정책 변경 사항 수신 시 어드민에 즉시 반영하는 프로세스를 정착시켜, 항상 최신 기준이 유지되는 운영 흐름을 구축했습니다.
Action 02
VOC 분석 기반 운영 가이드 전면 재정비
파트너 VOC를 분석해 정책 혼선이 반복되는 구간을 식별하고, 해당 항목을 우선 기준화했습니다. 상담사 전원이 동일한 판단 기준으로 응대할 수 있도록 운영 가이드를 전면 재작성해 팀 전체에 배포했습니다. 파트너사별 이슈를 유형화·분류하고, 어드민에서 직접 수정 처리하는 체계를 팀 내 표준 절차로 정립했습니다.
Action 03
신규 온보딩 가이드 정비 및 예외 케이스 문서화
파트너별 예외 케이스를 유형화해 문서화하고, 신규 담당자가 빠르게 기준에 맞춰 응대할 수 있는 온보딩 자료를 체계화했습니다. 안내 정확도 향상과 온보딩 시간 단축에 기여했습니다.

Outcome

결과

파트너 문의 반복률 감소 — 판단 체계 수립 후 동일 유형 파트너 문의 반복률이 줄고, 오안내·파트너 컴플레인이 감소했습니다.
어드민 데이터 정합성 개선 — 기준 정의 및 오기재 수정으로 데이터 정합성이 개선되어 정산 오류 방지에 기여했습니다.
신규 온보딩 기간 단축 — 체계화된 기준 자료와 예외 케이스 문서화로 신입 담당자 안내 정확도가 향상됐습니다.
400여 개 파트너 정책 기준화 완료 — 상담 판단 기준이 통일되어 응대 편차가 줄었습니다.

Relevance

커머스 플랫폼 운영에 연결되는 지점

다우기술 사방넷은 여러 쇼핑몰 파트너의 상이한 운영 조건을 하나의 플랫폼으로 연결하는 B2B SaaS입니다. 파트너별로 다른 정책을 수집·기준화하고, 상담 판단 체계를 통일하는 이 경험은 사방넷 운영 환경에서 파트너사 문의를 유형화하고 운영 기준을 정렬하는 업무와 직접적으로 연결됩니다. 400여 곳의 파트너를 다루며 기준 혼선 구간을 VOC로 식별하고 운영 가이드로 통일한 방식은, 여러 쇼핑몰 셀러의 운영 문의를 받는 환경에서도 동일하게 적용됩니다.

Insights

이 케이스에서 배운 것

데이터가 '있는' 것과 '기준화된' 것은 다릅니다. 어드민에 파트너 정책 데이터가 존재했지만, 그것이 운영 기준이 되려면 수집·검증·구조화·배포까지 이어져야 했습니다. 이 과정에서 중요한 것은 완벽한 기준을 한 번에 만드는 것이 아니라, VOC로 혼선 구간을 먼저 찾아 우선순위를 정하고 순서대로 기준화해 나가는 것이었습니다. 운영 현장에서 기준이 없으면 사람이 채우지만, 사람마다 다르게 채우기 때문에 결국 기준을 만드는 일이 운영의 핵심이라는 점을 확인한 케이스입니다.
